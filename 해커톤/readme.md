# 유채림팀 마지막 프로젝트 과제 및 발표자료
#### 팀원: 20184754 김현주, 20184487 유채림

### 제출 파일 설명
1. 발표자료_유채림팀.pdf : 프로젝트 발표자료
2. system.py: open cv와 여러 input,output 시스템을 이용한 마스크 집중관리 smart home 파일
3. hospital_api.py: api를 이용하여 선별진료소를 크롤링해주는 python 파일
4. mask_shopping.py: api를 이용하여 마스크 구매를 도울 수 있게 마스크 이름, 사이트를 크롤링해주는 파일 
원래 api를 가져온 것을 flask 서버에 올려서 스마트홈 사용자가 이용하게 하는 것이 목표였는데 시간이부족하여 api 크롤링한 것만 보냅니다!

### 프로젝트 설명
  코로나로 인해 모두가 마스크를 착용하고 외출을 해야한다. 하지만 아직 그것에 익숙해지지 않아서 마스크 착용을 깜빡하고 외출을 할 때가 꽤 있다.
  그런 사람들을 위하여 집 밖을 나가려 문 앞에 서면 마스크 착용여부를 자동으로 검사해주고 착용하지 않았다면 마스크를 올려둔 선반을 서보모터로 회전시켜 문 앞으로 오게 한다.
  이 프로젝트에선 집을 예시로 들었지만 지하철 개찰구, 음식점 입구 등 다양한 곳에 마스크를 착용하지 않았을 시 출입이 불가능하도록 검사할 수 있게 사용할 수 있다.
